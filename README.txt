DESCRIPTION - Describe the package in a few paragraphs
    Our website is implemented by html/css/javascript, and is deployed on Github pages:
        https://github.gatech.edu/pages/gni7/DVAWeData/
    Data are store in asset folder

INSTALLATION
    Due to our website implemented by pure html/css/javascript, no installation is needed.

EXECUTION
    # Our website is deployed to Github pages, the URL is:
        https://github.gatech.edu/pages/gni7/DVAWeData/
        
    # To watch our web page on your local computer, follow the following steps:
        Step 1: Go into DVAProject folder
        Step 2: Setup a local HTTP server in your terminal
            # If Python version returned above is 3.X
            python3 -m http.server
            # On windows try "python" instead of "python3"
            # If Python version returned above is 2.X
            python -m SimpleHTTPServer
        Step 3: Go to "http://0.0.0.0:8000/" on your browser, then you will see the web page
